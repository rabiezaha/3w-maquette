Au cours de cette formation nous allons avoir 2 matiees :

L'integration

Le HTML:

Le HTML sert a creer le contenu du dite : elle va contenir uniquement la partie textulle du site.

Le CSS:

Le CSS va nous permettre d'ajouter la couche graphique a notre site : les couleurs, les formes, le fond,etc.

le developpement

La Javascript :

Javascript est un langage de programmation de script employe pour rendre une page web interactive, le comportement de la page et son apparece pourra varier en fonction de certains parametres.

Le PHP :

Le PHP est langage de programmtion libre, utilise pour genere des pages web dynamyques,qui sont donc modifies en fonction de certines informations(heure,nom de l'utilisateur,formulaire remplir par l'utilisateur,formulaire remplir par l'utilisateur etc).

MySQL :

MySQL est un system de gestion de bases de donnees,il permet d'enrerigstrer des informations remplir par l'utilisateur (exemple :formulaire de creation de compte) pour ensuitr etre re-utiliser au moment approprie (ex:connexion a un site internet avec identifiant et mot de passe).
